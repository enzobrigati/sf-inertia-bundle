# Server-side adapter for Symfony 7 and Inertia.js v2

Thx to [Laravel adapter](https://github.com/inertiajs/inertia-laravel), [official Symfony bundle](https://github.com/SkipTheDragon/inertia-bundle).

## Installation

1. Add the required repository configuration to `composer.json`

```json
{
  "repositories": [
    {
      "type": "vcs",
      "url": "https://github.com/enzobrigati/inertia-bundle.git"
    }
  ]
}
```

2. Install the composer package

```shell
composer require enzobrigati/inertia-bundle
```
